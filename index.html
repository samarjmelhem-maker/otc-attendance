📋 Setup Instructions

1. Create Google Sheet:

  1. Go to sheets.google.com and create new sheet
  2. Name it "OTC Attendance System"
  3. Create 4 tabs: section1, section2, section3, active_codes
  4. Share → Anyone with link → Editor permission
  5. Copy the Sheet ID from URL

2. Get API Key:

  1. Go to console.cloud.google.com
  2. Create/select project
  3. Enable "Google Sheets API"
  4. Create Credentials → API Key
  5. Enter both in Settings above
